Welcome to my Page

JavaScript is the programming language of HTML and the Web. It allows websites to be interactive.

As a requirement to complete/pass our JavaScript course, our instructor, Professor Marin Rusi tasked us to create javascript programs. I put them into single page which provides the links(buttons) to the different programs for easier navigation. There are 5 navigation links at the side of the page.

  • Home - is the main page which provides description of the page and each section.
  • Section A - is a currency converter. It allows you to convert Canadian dollars to US dollars.
  • Section B - is a form. It allows you to enter information and post it.
  • Section C - is a calculator which allows you to multiply and divide numbers.
  • Section D - is a list box which allows you to select a season and display corresponding image.
